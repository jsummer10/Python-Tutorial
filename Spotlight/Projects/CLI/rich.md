# Rich

## Project

[https://github.com/Textualize/rich](https://github.com/Textualize/rich)

## What It Is

Rich is a Python library for creating formatted terminal output. It adds color, styles, tables, progress bars, panels, tracebacks, Markdown rendering, syntax highlighting, and other display features to command-line programs.

Plain terminal output is useful, but it can become difficult to scan once a program prints structured information. Rich gives Python scripts and CLI tools a way to present that information clearly without requiring a full graphical interface.

Rich is also the foundation for more advanced terminal interfaces in the Textual ecosystem. It works well on its own for better output, and it also pairs naturally with CLI frameworks like Click and Typer.

## What It's Used For

- Styled Console Output: Rich can print colored and styled text in the terminal.
- Tables and Panels: It can display structured data in readable terminal layouts.
- Progress Bars: Rich includes progress indicators for long-running tasks.
- Better Tracebacks: It can render tracebacks with syntax highlighting and local variables.
- Markdown and Syntax Highlighting: Rich can display Markdown documents and highlighted source code in the terminal.
- CLI Polish: Developer tools often use Rich to make command output easier to read and navigate.
