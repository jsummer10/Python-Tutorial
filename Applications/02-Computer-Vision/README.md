# Python Computer Vision

This folder contains applied computer-vision examples using OpenCV and sample
images.

## Examples

- [Intro](Intro/run.py) - loads an image, converts color spaces, and prints
  pixel-level information.
- [Convolution Filters](convolve/convolve.py) - applies OpenCV convolution
  kernels to an image.

## Try It

```bash
python Intro/run.py
python convolve/convolve.py
```

Install OpenCV, NumPy, and Matplotlib before running these scripts.
