# Python Tutorial

This repository is a hands-on Python tutorial organized as a set of lessons,
example programs, and small application demos. It is intended to be used as a
course companion or self-study reference, with most topics backed by notebooks
or runnable Python files.

## Repository Structure

- [Lessons](Lessons) - the main curriculum, split into beginner, intermediate,
  and advanced tracks.
- [Applications](Applications) - applied examples for machine learning,
  computer vision, and manipulating office documents.
- [Sample](Sample/PyTutorial) - a small Python package used for packaging and
  testing examples.
- [Spotlight](Spotlight) - one-off notes and project spotlights.

## How to Use This Repo

Start in [Lessons](Lessons) and work through the tracks in order if you are new
to Python. If you already know the basics, jump directly to the topic folder you
need. Many lessons include Jupyter notebooks, while framework and application
examples are provided as regular Python files.

Install dependencies as needed for the specific lesson or example you are
running. Some examples use third-party libraries such as Flask, Django, FastAPI,
PyQt5, OpenCV, Pillow, or testing and packaging tools.
