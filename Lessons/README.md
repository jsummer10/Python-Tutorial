# Python Lessons

This folder contains the main tutorial curriculum. The lessons are grouped by
experience level and are designed to be used in order, though each topic folder
can also stand on its own as a reference.

## Tracks

1. [Beginner](01-Beginner) - syntax, data types, control flow, functions,
   modules, files, documentation, and formatting.
1. [Intermediate](02-Intermediate) - object-oriented programming, data
   structures, iterators, decorators, regular expressions, CLIs, packaging,
   testing, static analysis, and config files.
1. [Advanced](03-Advanced) - concurrency, advanced OOP, data handling, user
   interfaces, web servers, image processing, and performance.

## Working With Lessons

Most topic folders include a `README.md` plus one or more Jupyter notebooks or
Python scripts. Open the README first for the topic map, then run the notebook or
script listed in that folder.
