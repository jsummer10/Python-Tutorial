# Python File I/O

This lesson covers reading from and writing to files with Python. The
`file_io_demo` folder contains sample text, CSV, JSON, binary, archive, and
configuration files used by the notebook.

## Lessons

1. [File I/O](01_file_io.ipynb)

## Topics

- Opening and closing files
- File modes, buffering, and encodings
- Reading and writing text and binary data
- Working with CSV, JSON, JSON Lines, and INI files
- Copying, moving, archiving, and safely replacing files
