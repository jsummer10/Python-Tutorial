# Python Modules

Modules help split programs into reusable files. This lesson introduces imports,
script entry points, and simple package layouts using the example files in this
folder.

## Lessons

1. [Modules and Packages](01_modules.ipynb)

## Supporting Files

- [mathutils.py](mathutils.py) - small utility functions for import examples.
- [shapes2d.py](shapes2d.py) - a single-file module example.
- [greet_script.py](greet_script.py) - a script-style module.
- [geometry](geometry) - a minimal package with submodules.

## Topics

- `import` styles and aliases
- How Python finds modules with `sys.path`
- The difference between modules, scripts, and packages
- Package `__init__.py` files
