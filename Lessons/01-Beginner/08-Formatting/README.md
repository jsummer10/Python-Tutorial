# Python Formatting

This lesson introduces formatting conventions that make Python code easier to
read and review. It focuses on PEP 8 and the practical habits that keep code
consistent across a project.

## Lessons

1. [Code Formatting and Style](01_formatting.ipynb)

## Topics

- Why code style matters
- PEP 8 naming and layout conventions
- Indentation and whitespace
- Line length and imports
- Formatting tools such as Black, Ruff, and isort
