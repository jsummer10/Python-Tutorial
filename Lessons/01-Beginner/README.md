# Python Basics

The beginner track introduces the core language features needed to read, write,
and organize small Python programs. Work through these sections in order if you
are new to Python.

## Sections

1. [Introduction to Python](01-Intro) - install Python, run code, and use
   Jupyter notebooks.
1. [Data Types](02-Data-Types) - numbers, booleans, strings, and common built-in
   containers.
1. [Control Flow](03-Control-Flow) - conditionals, loops, pattern matching, and
   loop control.
1. [Functions](04-Functions) - define reusable behavior with parameters,
   returns, scope, and flexible arguments.
1. [Modules](05-Modules) - split code across files, import modules, and build
   simple packages.
1. [File I/O](06-FileIO) - read, write, encode, serialize, and organize files.
1. [Code Documentation](07-Code-Docs) - write comments, docstrings, and type
   hints.
1. [Formatting](08-Formatting) - use PEP 8 and formatting tools to keep code
   readable.
