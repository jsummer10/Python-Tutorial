# Python Data Types

This lesson introduces Python's core built-in data types and the operations used
with them every day. It starts with basic scalar values and then takes a deeper
look at strings.

## Lessons

1. [Introduction to Data Types](01_basics.ipynb)
1. [Strings - A Deep Dive](02_strings.ipynb)

## Topics

- Checking values with `type()`
- Integers, floats, booleans, and `None`
- Strings, indexing, slicing, and common string methods
- Basic conversions between types
