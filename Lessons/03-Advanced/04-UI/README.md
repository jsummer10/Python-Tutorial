# Python User Interface

This section contains examples for building graphical and browser-based user
interfaces with Python. The examples range from standard-library Tkinter to
desktop GUI frameworks and web UI libraries.

## Examples

- [Tkinter](Tkinter) - basic widgets, layout, menus, toolbars, status bars, and
  dialog handling.
- [PyQt5](PyQt5) - widgets, signals, dialogs, main windows, file dialogs, and a
  calculator example.
- [NiceGUI](NiceGUI/app.py) - a browser-based task dashboard.
- [Shiny](Shiny/app.py) - a Shiny for Python task dashboard.

## Try It

```bash
python Tkinter/gui.py
python Tkinter/gui_project.py
python PyQt5/BasicGUI.py
python NiceGUI/app.py
shiny run --reload Shiny/app.py
```

Install the relevant GUI or web UI package before running examples that are not
part of the Python standard library.
