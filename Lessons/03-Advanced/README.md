# Python Advanced

The advanced track covers Python features and ecosystem libraries used in larger
applications, data workflows, user interfaces, services, and performance-focused
code.

## Sections

1. [Concurrency](01-Concurrency) - threads, multiprocessing, async patterns, and
   coordination concerns.
1. [Advanced Object-Oriented Programming](02-Advanced-OOP) - dunder methods,
   properties, descriptors, and richer class behavior.
1. [Data Handling](03-Data-Handling) - loading, inspecting, selecting, and
   transforming data.
1. [User Interface](04-UI) - GUI and web UI examples with Tkinter, PyQt,
   NiceGUI, and Shiny.
1. [Server](05-Server) - Flask, FastAPI, and Django examples.
1. [Images](06-Images) - image processing with OpenCV and Pillow.
1. [Performance](07-Performance) - timing, profiling, and optimization basics.
