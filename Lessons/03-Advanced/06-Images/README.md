# Python Image Manipulation

This section covers image processing with two common Python libraries: OpenCV
and Pillow. The notebooks show how to create, inspect, transform, and draw on
images.

## Lessons

1. [Image Processing with OpenCV](01_opencv.ipynb)
1. [Image Processing with Pillow](02_pillow.ipynb)

## Topics

- Loading and creating images
- RGB, BGR, grayscale, and color-space conversions
- Drawing shapes and text
- Resizing, cropping, and transforming images
- Choosing OpenCV or Pillow based on the task
