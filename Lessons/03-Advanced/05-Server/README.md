# Python Server

This section compares common ways to build web servers and web applications in
Python. The examples include lightweight API-style apps and a small Django
project.

## Examples

- [Flask](Flask/app.py) - routes for a home page, health check, greeting, and
  JSON addition endpoint.
- [FastAPI](FastAPI/app.py) - placeholder for a FastAPI example.
- [Django](Django) - a small Django project with an app, models, views, URLs,
  templates, migrations, and a SQLite database.

## Try It

```bash
python Flask/app.py
python Django/manage.py runserver
```

Install the framework you want to run first, such as Flask, FastAPI, or Django.
