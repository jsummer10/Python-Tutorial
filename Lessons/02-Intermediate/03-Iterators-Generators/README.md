# Python Iterators & Generators

Iterators and generators make it possible to process data lazily instead of
building every result up front. This lesson shows how Python's iteration
protocol works and how to write custom lazy pipelines.

## Lessons

1. [Iterators and Generators](01_iter_gen.ipynb)

## Topics

- Iterables versus iterators
- `iter()` and `next()`
- Custom iterator classes
- Generator functions and `yield`
- Generator expressions and lazy processing
