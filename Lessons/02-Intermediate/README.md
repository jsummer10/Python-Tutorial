# Python Intermediate

The intermediate track builds on the basics by introducing larger-program
design, common standard-library tools, command-line applications, testing, and
code-quality workflows.

## Sections

1. [Object-Oriented Programming](01-OOP) - classes, instances, methods, and
   inheritance.
1. [Advanced Data Structures](02-Advanced-Structures) - `collections`, heaps,
   sorted lists, queues, arrays, and memory views.
1. [Iterators and Generators](03-Iterators-Generators) - lazy iteration and
   generator functions.
1. [Decorators](04-Decorators) - wrapping functions and classes.
1. [Regular Expressions](05-RegEx) - pattern matching with the `re` module.
1. [Command-Line Interfaces](06-CLI) - argparse, Click, Typer, Rich, and
   Textual examples.
1. [Packaging](07-Packaging) - project structure, package managers, and delivery
   options.
1. [Testing](08-Testing) - unit tests, pytest, fixtures, and assertions.
1. [Static Analysis](09-Static-Analysis) - linters, formatters, type checkers,
   and coverage.
1. [Config Files](10-Config-Files) - YAML configuration loading and overrides.
