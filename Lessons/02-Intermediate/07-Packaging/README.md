# Python Packaging

This section introduces the tools and project structure used to turn Python code
into reusable packages and distributable applications.

## Package Managers

- `pip` - the standard Python package installer.
- `uv` - a fast modern tool for dependency management and project execution.
- `poetry` - a project and dependency manager with publishing support.

## Delivery

- `pyinstaller` - bundle Python programs into standalone executables.
- `briefcase` - package Python apps for desktop and mobile targets.
