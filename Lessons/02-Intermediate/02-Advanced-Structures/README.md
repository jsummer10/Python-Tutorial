# Python Advanced Structures

This lesson explores data structures beyond the beginner-level built-ins. The
focus is choosing the right structure for the problem and understanding the
tradeoffs each one brings.

## Lessons

1. [Advanced Data Types and Structures](01_advanced_structures.ipynb)

## Topics

- `collections` tools such as `Counter`, `defaultdict`, `deque`, and
  `namedtuple`
- Priority queues with `heapq`
- Maintaining sorted data with `bisect`
- Queues, arrays, sets, and memory-efficient views
