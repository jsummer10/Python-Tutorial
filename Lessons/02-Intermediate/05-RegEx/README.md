# Regular Expressions

Regular expressions are a compact language for finding and transforming text
patterns. This lesson introduces Python's `re` module and the most common regex
building blocks.

## Lessons

1. [Regular Expressions](01_regex.ipynb)

## Topics

- `search`, `match`, `findall`, and `sub`
- Character classes and quantifiers
- Groups and captures
- Flags such as case-insensitive matching
- Practical text validation and extraction
