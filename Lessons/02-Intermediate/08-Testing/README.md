# Python Testing

Testing checks that code behaves the way you expect and keeps working as it
changes. This section includes a notebook plus small `unittest` and `pytest`
examples.

## Lessons

1. [Testing](01_testing.ipynb)

## Supporting Files

- [sample.py](sample.py) - simple functions used by the tests.
- [test_unittest.py](test_unittest.py) - standard-library `unittest` examples.
- [test_pytest.py](test_pytest.py) - pytest examples.

## Try It

```bash
python -m unittest test_unittest.py
pytest
```
